# 历史偏离与人物反应规则

> 适用于《黄龙坠星》及五代十国穿越线。  
> 核心原则：历史是初始条件，不是固定剧本。主角介入以后，后续变化必须由人、利益、信息和制度共同推动。

## 1. 总原则

主角进入 943 年以后，历史不能照搬。

照搬会削弱故事，因为一个外来变量已经进入：他救过人、改变过船队、搅动过吴越海防、影响过闽国局势、带来了不同的卫生和组织方法。只要这些事情发生，后面的政局、人际、战役和人物判断都会产生偏移。

但偏移不能随便来。每一次改变都要回答四个问题：

1. 主角做了什么具体行动？
2. 这个行动改变了谁的利益、恐惧或判断？
3. 相关历史人物按自己的性格会怎么反应？
4. 这个反应会把局势推向什么新后果？

如果答不上来，就不要改大历史。

## 2. 历史节点的三种用法

### 初始条件

943 年以前发生的事，基本视为硬设定：唐亡、后梁、后唐、后晋、燕云十六州、吴越立国、南唐建立等。

这些事是主角进入世界时已经存在的地基，不轻易改。

### 压力事件

943 年以后原本会发生的大事，不必照搬，但要作为压力源存在。

例如：

- 后晋与契丹关系恶化。
- 闽国崩坏，南唐和吴越争夺福建。
- 吴越王位继承不稳。
- 后周兴起。
- 柴荣改革与南征北伐。
- 赵匡胤进入禁军体系。

这些事件可以变形、提前、延后、局部失败或产生新结果，但不能无缘无故消失。

### 分叉节点

当主角已经对某条线造成足够影响，大历史可以分叉。

例如：

- 吴越因主角改善海防，在福州局势中拿到更多筹码。
- 钱弘佐因主角医疗或政治介入，死亡时间发生变化。
- 胡进思政变被延后、失败，或以另一种形式爆发。
- 南唐因为主角破坏其海路，扩张节奏改变。
- 柴荣因主角提供医学或情报支持，北伐结果改变。

分叉越大，铺垫越要长，代价越要重。

## 3. 偏离等级

### 一级偏离：局部生活改变

范围：黄龙岛、一个船队、一个村落、一个小港口。

例子：

- 岛民死亡率下降。
- 海盗袭击失败。
- 一条新航线更安全。
- 某个虚构人物活下来或死去。

这种偏离可以早发生，适合第一卷。

### 二级偏离：地方权力改变

范围：明州、杭州、吴越海防、福州局势。

例子：

- 吴越提前重视海防和船坊。
- 某个旧将因利益受损开始反主角。
- 南唐提前派间谍调查黄龙岛。
- 闽地难民被主角组织起来，成为一支新力量。

这种偏离需要几章到半卷铺垫。

### 三级偏离：国家政策改变

范围：吴越、南唐、后晋、后周等政权路线。

例子：

- 吴越改变对南唐的军事策略。
- 钱弘佐或钱弘俶决定重用主角，但同时限制他。
- 李璟改变攻闽节奏。
- 后周对南唐战争出现新战法或新后勤体系。

这种偏离需要明确的政治利益交换和反对派。

### 四级偏离：王朝命运改变

范围：柴荣寿命、陈桥兵变、吴越纳土、南唐灭亡、宋是否出现。

例子：

- 柴荣未按原历史病逝，后周继续扩张。
- 赵匡胤不再通过陈桥兵变建宋，或建宋时间、方式改变。
- 吴越不再简单纳土，而是以盟约、自治或战争方式进入统一格局。
- 南唐提前衰败或短暂复兴。

这种偏离必须到第三卷以后再动。不能靠主角一句话改变，必须有医疗、军事、政治、后勤、人物信任等多条因果共同支撑。

## 4. 历史人物反应模型

### 钱弘佐

性格基调：年轻、好读书、温和，但不等于软弱；受制于旧臣和军功集团。

面对主角时：

- 会好奇，也会欣赏实用才能。
- 不会一开始就交大权。
- 会把主角放在海防、船坊、水利、防疫这类可控位置试用。
- 如果主角声望过快上升，他会既想保护又想限制。
- 他若被主角救命，信任会加深，但王府旧臣会更警惕。

合理偏离：钱弘佐可能因主角医疗或政治提醒多活一段时间，但这会直接冲击钱弘倧、钱弘俶、胡进思等人的命运，必须引发更大的吴越内斗。

### 钱弘俶

性格基调：务实、谨慎、保境安民，懂得小国生存之道。

面对主角时：

- 会重用，但不会把国家命运押给一个外来人。
- 如果主角能减少民间损耗，他会支持。
- 如果主角要吴越主动争霸，他会本能犹豫。
- 他最关心的不是名号，而是吴越百姓和钱氏宗族能否保全。

合理偏离：即使吴越变强，钱弘俶也未必会称帝争霸；他更可能要求更高自治、更体面归附、更少战祸，或者在统一势力之间极限周旋。

### 李璟

性格基调：有文化、有野心，继承南唐富庶家底，但容易被扩张欲和朝臣推动。

面对主角时：

- 不会把主角当神仙，而会当作吴越的新式人才或危险变数。
- 会派人试探、收买、离间，必要时刺杀。
- 如果主角影响福建局势，李璟会改变攻闽策略，而不是坐看吴越坐大。
- 他可以欣赏主角才华，但国家利益优先。

合理偏离：南唐可能因主角干预损失福建收益，转而更早攻击吴越海路；也可能试图把主角抢到金陵。

### 郭威

性格基调：军伍出身，务实，能体恤士卒和百姓，也深知兵权危险。

面对主角时：

- 会看重主角的组织和后勤能力。
- 不会轻信玄奇来历。
- 会先查背景、看结果、试忠诚。
- 如果主角能提升军需和民政，郭威会用他；如果主角动摇军心，郭威会压他。

合理偏离：主角可以影响郭威某些政策细节，但很难改变郭威的根本路线：先稳军心、再稳民生、再处理天下名分。

### 柴荣

性格基调：雄心极强、执行力强、有统一志向，不需要主角启蒙。

面对主角时：

- 会快速理解主角方案的价值和风险。
- 会追问成本、时间、兵源、粮秣、工匠、执行人，而不是只听愿景。
- 会欣赏主角，但也会试图把主角纳入后周体系。
- 如果主角不肯受控，柴荣会把他视为未来变量。

合理偏离：柴荣线是第三卷最大分叉点。主角如果延长柴荣寿命或改善北伐后勤，后周可能替代宋成为统一主体。但这会改变赵匡胤、赵普、吴越、南唐全部后续路线，必须谨慎推进。

### 赵匡胤

性格基调：军人魅力强，能聚人，懂分寸，擅长在局势成熟时抓住机会。

面对主角时：

- 早期会把主角当有本事但危险的人。
- 不会轻易向主角袒露野心。
- 如果主角在军中声望过高，他会保持距离。
- 如果主角能带来实利，他会合作，但不会做附庸。

合理偏离：赵匡胤可以和主角成为朋友、同盟、竞争者，甚至互相欣赏的对手。不要把他写成被主角随便收服的打手。

### 耶律德光

性格基调：草原帝国君主，军事强势，能利用中原分裂，但入主汉地治理困难。

面对主角时：

- 若听说主角存在，会先判断是否能为契丹所用。
- 他关心的是边疆、贡赋、军事优势和统治成本。
- 如果主角改善中原或吴越后勤，他会视作潜在威胁。

合理偏离：主角不可能靠嘴炮阻止契丹南下。只能通过情报、后勤、联盟、边防和政治成本影响契丹行动。

## 5. 改史的因果模板

每次要改变历史大事，先写出这个模板：

```text
原历史节点：
主角介入行为：
直接受益者：
直接受损者：
历史人物按性格的反应：
短期后果：
长期连锁：
必须付出的代价：
是否值得进入正文：
```

例子：钱弘佐未按原时间去世。

```text
原历史节点：947 年钱弘佐去世。
主角介入行为：提前改善宫中火灾后遗症、感染、饮食或药物处理，并提醒王府防范旧臣借病谋权。
直接受益者：钱弘佐、主角、部分改革派。
直接受损者：可能继位的钱弘倧、钱弘俶身边势力、胡进思等旧军功集团。
历史人物按性格的反应：钱弘佐更信任主角但不会完全放权；钱弘俶谨慎旁观；胡进思转入暗处；南唐加快离间。
短期后果：吴越政局暂稳，改革空间扩大。
长期连锁：原本 948 年政变不再按原方式发生，但旧军功集团会制造另一场危机。
必须付出的代价：主角被卷入王位政治，失去技术顾问的中立身份。
是否值得进入正文：值得，但要作为第二卷主轴之一。
```

## 6. 写作禁忌

- 不要让主角因为知道历史，就每次都押中。
- 不要让历史人物为了配合主角改变自己的利益和性格。
- 不要把“改变历史”写成单点事件，要写成连锁反应。
- 不要让改革只有好处，没有反扑。
- 不要让主角救下一个人却不改变相关继承、派系、婚姻、军权、财政关系。
- 不要把宋朝统一路线当必然终点。主角进入以后，宋可以出现，也可以不出现；但无论哪种，都要从人物和局势里长出来。

## 7. 本书后续默认路线

为了避免照搬历史，后续正文默认采用“渐进偏离”路线：

1. 第一卷只改局部命运：黄龙岛、海盗、船队、难民。
2. 第二卷改变吴越内部权力平衡和福州局势。
3. 第三卷让后周、南唐、吴越、赵匡胤都开始因主角而重新判断天下。
4. 第三卷末才决定是否动柴荣寿命、陈桥兵变、宋朝建立方式等最高级分叉。

这条路线能保留真实历史的厚度，又不会把故事写成历史年表复述。
